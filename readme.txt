== Description ==  
AG AVENUE is a modern and responsive WordPress theme specifically designed for hardware stores and retail businesses. With a clean layout and intuitive features, AG AVENUE is perfect for showcasing products and creating a professional online presence for your store.  

Features include:  
- Responsive design optimized for all devices  
- Easy customization through the WordPress Customizer  
- Integrated WooCommerce support for eCommerce functionality  
- Pre-designed sections for featured products, services, and testimonials  
- SEO-friendly structure to enhance search engine visibility  

== Installation ==  
1. Go to **Appearance > Themes** in your WordPress dashboard.  
2. Click **Add New** and then **Upload Theme**.  
3. Choose the AG AVENUE ZIP file and click **Install Now**.  
4. Once installed, click **Activate** to start using the theme.  

== Frequently Asked Questions ==  

= Does AG AVENUE support WooCommerce? =  
Yes, AG AVENUE is fully compatible with WooCommerce, allowing you to create a complete online store.  

= Can I customize the colors and fonts? =  
Absolutely! AG AVENUE comes with customization options for colors, fonts, and layouts via the WordPress Customizer.  

= Is the theme translation-ready? =  
Yes, AG AVENUE supports translation files, making it easy to use in multiple languages.
